# Vulktech Website

Landing page for Vulktech - a product development company focused on shipping ideas fast.

## Tech Stack

- HTML5
- Tailwind CSS (via CDN)
- Inter font (Google Fonts)

## Features

- Responsive dark-themed design
- Custom brand colors (vulk-dark, vulk-blue, vulk-graphite)
- Social media links (X, LinkedIn, Instagram, Linktree)
- Contact CTA linking to email

## Getting Started

Simply open `index.html` in a browser or serve it with any static file server.
